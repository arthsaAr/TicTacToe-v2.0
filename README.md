# Tic-Tac-Toe Android Game

Classic Tic-Tac-Toe game built with Unity and C# for Android devices.

## Features

- Touch-optimized controls for mobile gameplay
- Win/draw detection with visual feedback
- Score tracking across multiple rounds
- Clean game state management
- Installable Android .apk

## Download & Install

**For Android Users:**
1. Download the `.apk` file from [Releases](https://github.com/arthsaAr/TicTacToe-v2.0/releases)
2. Enable "Install from Unknown Sources" in Android settings
3. Install and play!

**For Developers:**
```bash
git clone https://github.com/arthsaAr/TicTacToe-v2.0.git
# Open in Unity and build for Android
```

## Tech Stack

- Unity Game Engine
- C# (object-oriented design, state management, event handling)
- Unity UI system
